問題5-1 EUの旗を描画する

星を描く関数を作り、それを用いてEUの旗を描画してください。

ステップ 1 / 5
まずは円形に12個の〇を描画するところまで
function setup(){
  createCanvas(200, 200);
  background(192);
  noStroke();
  for(let i = 0; i < 12; i++){
    const theta = TWO_PI * i / 12;
    const x = 100 + cos(theta) * 50;
    const y = 100 + sin(theta) * 50;
  }
}

問題5-2 吹き出しを描画する

文字列を吹き出しに入れて描画する関数 function balloon(t, x, y) を完成させてください。

  • 要件1 引数 t で吹き出し内に表示する文字列を指定できるようにする
  • 要件2 引数 x, y で場所を指定できる（吹き出しの左上座標になる）ようにする
  • 要件3 吹き出しには背景色を付ける（お好みの固定色でよい）
  • 要件4 吹き出しにはしっぽ？を付ける（お好みの形でよい。たとえば吹き出しの下中央から三角形を伸ばすなど。）

余力がある人は、吹き出しの形などを引数で指定できるようにしてみるとよいでしょう。

問題5-3 カレンダーを描画する（チャレンジ問題）

以下の要件を満たすようにカレンダーを描画する関数 drawCalendar(y, m) を完成させてください。 難しい問題になりますので、できたところまでで構いませんが、すぐには諦めないで挑戦してみてください。

  • 要件1 西暦y年が何日あるかを返す関数 daysInYear(y) を完成させる。条件分岐一発で結果は365か366になります。
  • 要件2 西暦y年m月d日が何曜日かを返す関数 dayOfWeek(y, m, d)。結果は0(=日曜日), 1(=月曜日), ... 6(=土曜日)としましょう。
    • ヒント：曜日がわかっている基準日を1つ決めて、その基準日からの日数を7で割った余りを計算する方針で作れます。 基準日は何でもよいのですが、たとえば1970年1月1日とかにします。
    • 完成したらいくつか具体的な日付を引数として実行してみて合っているか確認しましょう。たとえば dayOfWeek(1981, 7, 17) は5（金曜日） dayOfWeek(2001, 6, 25) は1（月曜日）です。
  • 要件3 関数 drawCalendar(y, m) は引数で与えられた年月のカレンダーを描画する。
  • 要件4 日曜日始まりで、日曜日は赤、土曜日は青で描画する。
