問題4-1 棒グラフ

棒グラフを描くプログラムを作りましょう。

  • 要件1 配列に含まれる値の平均値を求める
  • 要件2 配列に含まれる値のうちの最大値・最小値を求める
  • 要件3 最高・最低の棒を赤・青に色分けする
  • 要件4 平均値の線を緑色で引く

ProgTouch を使って全体の流れを確認してから取り組みましょう。 ステップ数が多く、コードが長くなってスクロールが発生します。全体像も把握しながら進めてください。

ステップ 1 / 12
配列の中身を乱数で初期化。背景の目盛り線を先に引いておこう。
function setup(){
  createCanvas(200, 200);
  background(240);


  // 配列をランダムに初期化する
  let scores = [];
  for(let i = 0; i < 10; i++){
    scores[i] = random(20, 100);
  }
  console.log(scores);
  
  
  // 背景に横線をn本引く
  const n = 10;
  for(let i = 0; i < n; i++){
    line(0, height * i / n, width, height * i / n);
  }
  
}
