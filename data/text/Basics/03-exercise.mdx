# 演習問題「アニメーションとインタラクション」

### 問題3-1 確認問題
心臓の鼓動のようなアニメーションを作ってみましょう。

- 要件１：円が周期的に大きくなったり小さくなったりする
- 要件２：キーを押していると鼓動が早くなって離すとゆっくりになる

ボールのアニメーションにおける画面端処理と考え方としては似ているところがあります。
行ったり来たりするのが位置ではなくてサイズになるだけで、
小さくなるときと大きくなるときを条件分岐で切り替えることで実現できます。

周期的なアニメーションにする方法についてのコツは以下の ProgTouch で学んでいきましょう。

<ProgTouchEmbed>
<Step thoughts="まずは円を描くところまで。サイズは後で変えられるように変数にしておこう。">
```javascript
// @A
let size = 50;

function setup() {
  createCanvas(200, 200);
}

function draw() {
  background(160, 192, 255);
  // @B
  ellipse(width / 2, height / 2, size, size);
}
```
</Step>

<Step at="A" thoughts="周期的なアニメーションは1周期のフレーム数と何フレーム目かを覚えるようにする">
```javascript
const cycle = 100; // 1周期のフレーム数（定数）
let count = 0; // 現在のフレーム数（変数）
```
</Step>

<Step at="B" thoughts="毎回の draw でフレーム数を進める">
```javascript
  count = (count + 1) % cycle;
```
</Step>

<Step at="B" thoughts="周期の前半は大きくして、後半は小さくする">
```javascript
  if (count < cycle / 2) {
    size += 1;
  } else {
    size -= 1;
  }
```
</Step>

<Step undo={1} thoughts="違う書き方もしてみよう" />

<Step at="B" thoughts="あるフレームでのサイズを直接計算する方法もある　どちらが好み？">
```javascript
  // @C
  if (count < cycle / 2) {
    size = 50 + cycle - count;
  } else {
    size = 50 + count;
  }
```
</Step>

<Step at="C" thoughts="キーを押しているときはさらにフレームを進める（実行結果をクリックしておかないと反応しません）">
```javascript
  if (keyIsPressed) {
    count = (count + 2) % cycle;
  }
```
</Step>

### 問題3-2 2Dアクションゲームのようなインタラクション
昔ながらのマリオのような2Dアクションゲームのキャラクター移動を模したインタラクションを作ってみましょう。

- 要件１：左右キーで地面の上を左右に移動
- 要件２：左右キー＋好きなキーを押している間だけ移動速度アップ
- 要件３：スペースキーでジャンプ（空中ジャンプ不可）

要件１と要件２はテキストの「キーボード操作に反応する」のサンプルほぼそのままです。

要件３はテキストの「重力、床に弾む」の応用になります。
地面で跳ね返る代わりにキー操作でジャンプ（vx を大きめの負の値にする）するようにしてください。
そのままだと空中でもジャンプできてしまうので、地面にいるときだけジャンプできるように if 文を追加しましょう。
